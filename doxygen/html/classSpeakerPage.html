Detailed Description

A component that allows the selection and editing of on ore more speakers.

The component consists of two parts: A SpeakerList , allowing to select various speakers, and an AllowedRolesEditor which allows the editing of the selected speakers properties. Additionally, a TeamEditor is used which enables the editing of teams.
