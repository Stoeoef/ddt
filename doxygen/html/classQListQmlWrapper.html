Detailed Description

Allows accessing a cpp list in QML.

QML cannot handle lists of type QList. This class fills this gap by offering a list like interface on the cpp side while inheriting QAbstractListModel, which is neccessary for elements like the QML ListView. A QListQmlWrapper can be set directly as a ListView's, model, the model item roles are specified by a concrete subclass of . See roleNames for details.
Deleting an item will automatically remove it from the wrapper's list.
