Detailed Description

Allows editing multiple speakers at once.

MixedValueSpeakerModel has an interface similar to SpeakerModel, it contains all its boolean typed properties. It accesses a list of different SpeakerModel and combines their properties. If the list contains different values for some property, PropertyState::MIXED will be returned as the property's value. Moreover, setting some property for all speakers at once is possible.
