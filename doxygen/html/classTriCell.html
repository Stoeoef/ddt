Detailed Description

A styled cell with three different possible states.

This component is functional similiar to a tri-state check box. The states are indicated with three different colors, the cell has a hover animation. The user can read and set the component's state with the state property. Three values are allowed: -"off": indicates that the cell is inactive -"on": indicates that the cell is active -"mixed": indicates that the cell state is unknown
